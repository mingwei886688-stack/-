📖 使用方式

  1. 点击上方按钮下载对应系统的压缩包
  2. 解压到任意目录，双击「文件大助-Web」运行
  3. 浏览器会自动打开，在网页里操作即可
  4. 所有数据处理都在本地，不上传任何内容
  5. macOS 首次打开若提示「无法验证开发者」，请去系统设置允许
